Operator Setup Guide
Complete these steps to enable live email functionality for a mailbox account.
1
Create a mailbox in cPanel
Log in to your cPanel account, navigate to Email Accounts, and create or identify the mailbox you want to connect. Note the full email address.
2
Note the IMAP host and port
In cPanel, find the IMAP settings for the account. Default values are usually mail.yourdomain.com port 993 (SSL) or 143 (STARTTLS).
3
Note the SMTP host and port
SMTP settings are typically mail.yourdomain.com port 465 (SSL) or 587 (STARTTLS). Confirm with your hosting provider.
4
Add credentials to server environment
Add the following environment variables to your server: EMAIL_IMAP_HOST, EMAIL_IMAP_PORT, EMAIL_SMTP_HOST, EMAIL_SMTP_PORT, EMAIL_USER, EMAIL_PASS. Never commit credentials to source control.
5
Set account status to "active"
Once credentials are in place and connectivity is verified, edit the mailbox account record on this page and change its status from pending to active. The system will then attempt to use those credentials for email operations.
