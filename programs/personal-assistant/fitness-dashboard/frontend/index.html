📱 Setup — Health Auto Export
  1. Install Health Auto Export on your iPhone (App Store, Premium required)
  2. Open the app → tap Automations → + New
  3. Select REST API as the export type
  4. Paste the webhook URL shown below into the URL field
  5. Set Method to POST, Format to JSON, Version to V2
  6. Enable Workouts and optionally Metrics
  7. Set Schedule: Every 1 hour or On Charge
  8. Tap Save and run once manually to test
Tip: Enable "Include Route" and "Include Heart Rate Data" in the workout options for full XP bonuses.
